Sets up the basic typography for your project. Make sure to set a $breakpoint-default in the settings and populate the $types starting with your default breakpoint. After that the only thing that needs to be defined is whatever differs for each breakpoint.

Font mixin

@include type(title); / .t-title

// setup variable $types: ( desktop: ( default: ( font-size : 1rem, font-family : $t-default-font-family, ), title: ( font-weight : $t-bold, font-size : 1.75rem, line-height : 1.2 ), paragraph: ( font-size : 1rem ) ), mobile: ( default: ( font-size : 1.25rem, ), title: ( font-size : 3.5rem ) ) ); // include in your classes .title--content { @include type( $name: title, // name in your typo map $overwrites: (margin: 0) // overwrites ) } // or renders a helper class per item in $typo @include textClasses(".t-");

The horizontal mixin places things next to each other. It has to be applied to the parent element. The direct child of the element can be specified using the $childSelector argument. The different flavors are; inline (default), samewidth and equidistant
